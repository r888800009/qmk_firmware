# KB Paradise V60 Type R r888800009 Layout

Terminal and vim like

### Build
``` bash
make v60_type_r:r888800009:flash
```

### Lock Fn
two taps right fn key

### sent REISUB

1. hold <kbd>fn</kbd> + <kbd>menu</kbd>, and tap <kbd>alt</kbd> and <kbd>i</kbd> (SysRq) 
lock the two keys.
2. <kbd>reisub</kbd>
